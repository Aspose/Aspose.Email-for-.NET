==========================================
Aspose.Email for .NET Examples Read Me
==========================================

This package contains C# and VB.NET ASP.NET examples of Aspose.Email for .NET.

The solution files of Microsoft Visual Studio 2008 and 2010 are provided. 

Please open the solution file (.sln) according to the version of Visual Studio installed on your system.

Aspose.Email.Demos.2008.sln - Open with Microsoft Visual Studio 2008
Aspose.Email.Demos.2010.sln - Open with Microsoft Visual Studio 2010


How to Run the Examples
==========================================

Open the appropriate solution file (.sln) in Microsoft Visual Studio. Click on "Debug" menu and choose one of the following menu items:

- Start Debugging
- Start Without Debugging

Alternatively, you may also open any aspx page from the Visual Studio Solution Explorer, right click and choose "View in Browser" from the context menu.


Software Requirements
==========================================

- Aspose.Email for .NET 1.0.0.0 or later
- Aspose.Pdf for .NET 3.7.0.0 or later
- Any of the following Internet Browser
  - Microsoft Internet Explorer 6 or above
  - Mozilla Firefox 3 or above
  - Google Chrome
- Any of the following versions of Microsoft Visual Studio
  - Microsoft Visual Studio 2008 SP1 (Express, Standard, Professional and Team Editions)
  - Microsoft Visual Studio 2010 (Express, Professional, Premium and Ultimate Editions)


IIS Requirements
==========================================

All examples can run with the ASP.NET Development Server installed with Visual Studio.

You can configure IIS 5.1 or above to run the Aspose.Email Asp.NET examples.
  - Please Open IIS Management Console and add a new Application.
  - In "Alias", give "aspose.Email.demos" or any name of your choice.
  - In "Physical Path", give the complete path to the demos folder.
  - Run the examples in the browser using the IIS path settings e.g. [http://devsys01/aspose.email.Demos/default.aspx.]
